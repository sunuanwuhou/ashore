# 五人

```java
某公司招聘时有张三、李四、王五、赵六、钱七5人入围。
从学历看，有2人为硕士、3人为博士；
从性别看，有3人为男性、2人为女性。
已知，张三、王五性别相同，而赵六、钱七性别不同；
李四与钱七的学历相同，但王五和赵六的学历不同。
最后，只有一位女硕士应聘成功。
由此可以推出，应聘成功者为：
    A、张三 B、李四 C、 王五  D、赵六
```

我认为是运用了【并列关系】

```jaba
张三、王五性别相同  排除AC
李四与钱七的学历相同 排除B 选D
```
