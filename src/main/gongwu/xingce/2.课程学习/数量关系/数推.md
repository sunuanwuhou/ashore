# 做差-考点最多

+ 题目特征

  + 整个数列的递增不明显 幅度小（ 2-3倍）

  + 根据等差性质快速判断

    ![image-20231129083049601](.images/image-20231129083049601.png)

    大约实在3到5倍之间 a3-a2=n  a4-a1 =3n

+ 关注
  + 质数列 1 3 5 7 11 13 17 19 
  + 平方差

+ 解决方法

  + 如果满足三倍关系，直接利用计算
  + 否则
    + 优先考虑做差
    + 考虑二次做差
  + 如果()在中间，可以利用【隔项差】来实现
