# 基础知识

a 是最外层每边人数

+ 方正总人数=a*a
+ 方正最外层总人数=4a-4  【需要减去顶点重复的】
+ 最外层和内层相差= 每边相差2*4=8
+ 方阵减少一行一列，剩下和原来的还是平方数，直接秒杀平方数
