# 资料分析

[具体学习资料](../2.课程学习/2.资料分析)


1. 看题目问的时间，是不是跟材料一样。注意单位 年 月份 ！！！

   > 1. 有的题目会出现年份不同的情况 比如题目问的1-3 材料给的1-4
   > 2. 图标 柱状图注意对应的名字
   > 3. 2010年末就是2011年初

2. 日常坑

    1. 倍数

       > 1.是几倍数=A/B
       >
       > 2.是几倍=增长率+1
       >
       > 3.多几倍数=A/B - 1

    2. 增长率

       > 1. 高百分之几 是问增长率

3. 看题目问的关键词在材料中找到出处

4. 掌握常用技巧

   > 1. 尾数 21法 百化分 拆分 估算 截位直除 假设分配 份数法 增长趋势
   > 2. 柱状图问某某年同比增长超过百分之10的 可以用基期* 百分之10在与本期相比
   > 3. 均前每后是分母

5. 分清题目求的是什么，现有资料分析无非就是2种题型

    1. ABRX

        1. A : B/(1+r)  =B-X 【推荐错位加减 比列完美份数法】

            1. 基期差： 特征：一正一负 一大一小  方法：正负看现期差，且选项大概不等于现期差 [分析](../2.课程学习/资料分析/基期差计算技巧.md)

        2. B: A x (1+r)

        3. R

            1. 比较2个R趋势大小  【增长趋势比较法】

            2. 柱状图一排R趋势大小

               > 看本期增长趋势，看增长幅度趋势 然后把特殊的找出来进行比较

            3. 间隔增长率： 【R=r1+r2+r1*r2】

               > 现在一般都是给一个R和r1 求r2

            4. 年均增长率：【基期×（1+r）^n =现期】

            5. 混合增长率：

               > 特征：凡是a=b+c都可以用混合增长率
               >
               > 解答:   十字相乘 整体的增长率介于两部分增长率之间，且靠近基期较大的一方
               >
               > 列子：盈余=收入-支出 收入=支出+盈余 收入5 支出1.8 盈余肯定是大于5 如果要求具体值可用十字相乘或者线段法

            6. 比值增长率：

               > 特征：选项出现百分比点是比重 %是比值
               >
               > 解答：(a-b)/(1+b)
               >
               > 、

            7. 乘积增长率

               > 特征：凡是a=b*c都可以用乘积增长率
               >
               > a=b+c+ b*c

            8.

        4. X

            1. 直接求x

               > 1. 当R小于10%时，X约等于B*R
               > 2. 直接用错位加减或者假设分配
               > 3. 其他情况直接用截位直除法

            2. 比较2个增长量

               > 1. B越大 R越大则X越大
               > 2. 我的B是你的2倍，你的R至少是我的2倍以上，x才有可能相等

    2. 比重问题-考点最多
        1. 现期比重：【A/B】
        2. 隔期比重：【小集合 × 中集合】
        3. 基期比重：【A（1+b%）/B（1+a%）】 -考点最多
            1. 题型变种：  A B C 给 b c的增长率。求基期b/a的比重 先求出基期b/c的比重。在置换出b/a
           2. 快速计算基期比重：现期比重+-[a-b]  [分析](../2.课程学习/资料分析/基期比重快速技巧.md)
        4. 比重差：【（A/B） * （a-b）/（1+a）】

6. 遇到复杂题目，直接放弃，性价比不高




# 言语理解

[学习资料](../2.课程学习/1.言语理解)

## 逻辑填空

- 用词习惯：固定搭配

  > 1. 首先要排除的，就是不符合用词习惯的词，也就是不搭配的词 只需要提炼出主/谓/宾
- 词义侧重：词语的词义强调点是否符合逻辑提示，上下文对应
- 程度轻重：扫一眼首尾句，大致判断是否是总分结构，然后快速确定是否转折/递进/并列。
- 看全文态度，是褒义、中性、贬义
- 对比择优

## 中心理解

- 先读问题，再看题目

- 做好圈画，准确把握主题词（就是文章反复出现的关键词）

- 关注中心语句，强调句，转折句，原因句等语法结构，忽略举例，背景句，解释句等

- 看选项，一般是用排除法。对比择优

  > 1. 过于绝对
  > 2. 过于片面
  > 3. 无中生有
  > 4. 偷换概念

## 语句排序

- 读选项：筛选头尾； 逻辑顺序、时间顺序
- 读句子：确定抱团； 看是否有相近话题可以构成捆绑
- 代入检查

## 其他



+ 对策
    1. 重在说明是意图题，优先选对策。
    2. 优先选有具体表述的，空泛的不要选。
+ 语句衔接

    +  空格前有“因此”一词，其前是逗号，这表明结合最后一句即可得到答案。 其前是句号，是对全文的总结
+ 意图题

    +  一般是看最后一句话

## 图形推理


+ 常见规律

    + 有内有外，考察内外关系的数量 比如对称轴、面等
+ 对于分组的，看图形共性，有没有相同的属性
    + 九宫格优先横着看（横竖S米）、16宫格内部黑快数量不变，优先考虑回字型
+ 汉字三板斧： 笔画、封闭、结构(横竖撇捺)
    + 数字： 笔画、封闭、结构、对称
+ 封闭区间多 优先封闭空间
    + 曲直出现：考曲直之间的数量关系。交点、曲><直
+ 万般无奈一笔画：日字、田字、 多端点、 十字、圆的切点 优先考虑笔画数
+ 黑白球

    + 分组： 对称 笔画 部分  个数 
    + 找规律：位置移动  相邻比较 行变化



+ 元素组成相同，优先考虑位置规律

    + 动态
        + 平移
            + 特别注意，是可以斜着移动的 这种主要是找关键图形的变动！！
        + 翻转：时钟法
    + 静态：相交/相切/外离等

+ 元素组成相似，优先考虑样式规律

    + 加减同异
    + 样式遍历
    + 黑白相加、元素置换、自定义规律
    + 元素换算：要么递增、要么递减 关键是找包含

+ 元素组成不同，优先考虑属性规律

    + 对称 ：
        1. 图形出现箭头、“Z”字变形图、等腰三角形等，优先考虑对称性
        2. 对称轴 数量（面或者其他z）、方向、与点或者线重合
    + 曲直
    +  凹凸
    + 封闭

+ 元素组成不同，无明显属性规律，优先考虑数量规律

    + 线：线的方向

    + 面：最大面、面积、面和外框、相同面

##  定义判断

1. 看清题目问的是属于哪个定义，出题人会给多个定义

2. 最重要的考点是“主体/对象”，因为紧接着“A是”出现，很容易被忽略，要首先标注出来。 一个定义里，完整包含举例、特点、前提、目的（一般也就1、2个），一但出现，基本上就是考点。所以首先圈出“主体/对象”，然后再搜索“举例/特点/前提/目的”，迅速标注，对照选项。


## 逻辑判断

+ 常规翻译:
    + 掌握常规A->B 以及到 -A或B的转换 还有二难推理
    + 肯前必肯后，否后必否前 -> 其实就是逆否公式
    + 否前肯后，无必然结论
+ 形式推理：下列xx结构相似的是？
    +  直接写A->B的形式，然后套选项
+ 真假推理：
    + 利用矛盾关系
        + A->B矛盾命题为 A且-B
        + 反对关系
        + 有的有的必有一真、所有所有必有一假、
    + 包含关系
        + A→A或B 单独的分支可以推出含有的分支
        + 一真前假  一假后真
    + 最大信息
    + 一个人干一件事
    + 万般无奈假设、代入法

# 数量关系

## 数字推理

- 整个数列的递增不明显 幅度小（ 2-3倍）
  - 优先考虑做差
  - 考虑二次做差
- 递推数列：**是指前后项存在着一定的关系，然后按这个关系递推出结果，也就是后面的数是根据前面的数按照一定的关系得来的，最基本的递推关系就是和、差、积、商**
- 幂次数
  - 一般出现在有几个数字，突然冒出一个分数，这时需要考虑负幂次方变换。
  - 关注常用的立方数 27 64 125 216
- 分数
  - 化同：就把整个数列的分子或者分母化成同一个数，然后找规律。
  - 反约分：**可以把分母通分成一样的最好，不能的话就把就需要我们“造”一个数列出来，一般是分母和分子分别弄成一个明显关系的数列**。
  - 前后关系：顾名思义，这类数列前后项之间是有一定的关系的，一般是乘或者除，作差作和之类的比较少。
- 分组：
  - 题干一般都有**8个或8个以上**的数，又或者有明显**小数点**的
  - **奇偶数位分组、数字内部分组或数字合并分组等。**
- 机械拆分
  - 这些题目有个特征，就是**数与数之间毫无特征**，有时**突然增大或突然增小**，或者**都是很大的数**。
  - 数字内部规律一般是**内部相加减或相乘**之类的，行内一般叫机械拆分
- 画图
  - 如“九宫格”“四宫圆”等
  - 一般解法都是相邻数之间找关系，不外乎加减乘除，或者平方之类的。做这些题首要的是找到对应“？”所在位置的关系。
