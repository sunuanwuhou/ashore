   2. 全体人民共同富裕
   3. 物质文明和精神文明相协调
   4. 人于自然和谐共生
   5. 和平发展道路
3. 现代化本质要求/标准:

   1. 坚持党领导、社会主义
   2. 五位一体
   3. 建人类命运共同体、创建人类文明新形态
   4. **坚持中国共产党领导，坚持中国特色社会主义，实现高质量发展，发展全过程人民民主，丰富人民精神世界，实现全体人民共同富裕，促进人与自然和谐共生，推动构建人类命运共同体，创造人类文明新形态**

   4. 坚持深化改革开放
   5. 坚持发扬斗争精神

## 高质量发展

高质量发展是全面建设社会主义现代化国家的**首要任务**

+ 发展是党执政兴国的第一要务。没有坚实的物质技术基础，就不可能全面建成社会主义现代化强国
+ 高质量发展的必有之路:**加快实现高水平科技自立自强**
+ 高质量发展的战略基点:加快构建新发展格局(国内大循环为主体、国内国际双循环相互促进)
+ 高质量发展的必然要求:推进农业现代化
+ 高质量发展的最终目的:人民幸福安康

1. 质**的有效提升**和量的**合理增长**
2. 充分发挥**市场在资源配置中**的决定性作用，更好发挥**政府作用**
3. 的着力点放在**实体经济**上
4. 全面推进乡村振兴。全面建设社会主义现代化国家，最艰巨最繁重**的任务仍然在农村** 【百千万是县域】
   加快建设农业强国，扎实推动乡村产业、人才、文化、生态、组织振兴全面落实粮食安全党政同责，牢牢守住**十八亿亩耕地红线**
